Lời kết

Bootstrap là gì?

Bootstrap là một framework (mã nguồn mở) bao gồm các HTML. CSS và JavaScript. Trong bootstrap có các thành tố có sẵn như typography (kiểu chữ), forms (biểu mẫu), buttons (nút), tables (bảng), grids (bản mô tả ngắn), navigation (điều hướng trang web), image (ảnh),... tóm lại là các template (mẫu có sẵn) dùng để thiết kế và tối ưu website chuẩn responsive.

Định nghĩa bootstrap

Từ các mẫu sẵn có trong bootstrap các designer web có thể sáng tạo và phát triển tạo ra nhiều mẫu website hoàn chỉnh khác nhau với màu sắc, kích thước, hình ảnh,.. mới mẻ và sáng tạo hơn. Bên cạnh việc tiết kiệm thời gian khi quản lý các template web, mục tiêu chính của bootstrap là tối ưu hiển thị website giúp website chuẩn responsive.

Responsive Web Design là gì?

Responsive web là một website có thể tự đồng điều chỉnh kích thước để hiển thị vừa vặn trên tất cả các thiết bị truy cập website, từ màn hình nhỏ như điện thoại đến máy tính bảng hay màn hình lớn như máy tính, tivi web đều hiển thị đầy đủ không bị cắt góc ảnh hay mất chữ. Việc dùng bootstrap để tối ưu này giúp các nhà thiết kế web không phải tạo nhiều kích thước web khác nhau để tương thích với mỗi giao diện. Đặc biệt các mã nguồn mở bootstrap còn hoàn toàn miễn phí nên chi phí và thời gian thiết kế web cũng được tối ưu nhiều hơn.

Tính năng chính và cấu trúc của bootstrap

Về mặt cấu trúc, bootstrap bao gồm các tập tin JavaScript, CSS và fonts đã được dịch và nén lại, một số bootstrap còn được thiết kế dưới dạng module (mô -đun). Chính vì vậy nó tương thích với hầu hết các mã nguồn mở hay được dùng để thiết kế website như Wordpress, Joomla, Magento,... Về tính năng, Bootstrap nổi bật với các tính năng như - Cho phép designer tùy chỉnh mã nguồn mở của website trước khi tải xuống và sử dụng nó tại trang web cần thiết kế. - Các thành phần trên website được lưu và có thể dùng đi dùng lại nhiều lần - Bootstrap có sẵn các thành tố về giao diện, kiểu chữ, phông chữ, nút điều hướng,.. vô cùng đa dạng và cho phép designer dùng miễn phí để thiết kế giao diện website - Bootstrap cho phép tạo lập website dễ dàng với việc chỉ cần khai báo tính năng website do được tích hợp JQuery. - Định nghĩa glyphicons của bootstrap làm tối ưu việc dùng hình ảnh làm biểu tượng trang từ đó tăng tốc độ website

Lý do nên dùng bootstrap

Hiện nay có cả nghìn ứng dụng để thiết kế website nhưng các designer web vẫn ưu tiên sử dụng Bootstrap do những ưu điểm nổi bật như sau

  • - Dễ dàng thao tác
  • - Tiết kiệm thời gian
  • - Thành phẩm website chất lượng cao
  • - Độ tương thích của website với thiết bị sử dụng cao

Dễ dàng thao tác

Cơ chế hoạt động của Bootstrap là các mã nguồn mở nên người dùng có thể phát triển các thiết kế dựa trên bản ghi gốc. Chính vì tính mở như vậy nên các thao tác thiết kế với bootstrap vô cùng dễ dàng chỉ cần designer có các kiến thức cơ bản về 3 mã HTML. CSS, JavaScript.

Tiết kiệm thời gian

Bootstrap có các đoạn code sẵn với tính năng được lập trình sẵn nên các designer chỉ cần tải xuống hoặc copy để áp dụng cho website của mình. Chính vì vậy giúp tiết kiệm rất nhiều thời gian so với việc tự viết các đoạn code để tạo nên website.

Thành phẩm website chất lượng cao

Bootstrap không khiến cho giao diện website trở nên chắp và và rời rạc, nó chỉ tiết kiệm thời gian hơn cho các designer khi tạo ra giao diện web. Các thiết kế được designer tùy chỉnh và phát triển lại nên thành phẩm website sẽ có chất lượng cao và hoàn thiện hơn.

Tối ưu hiển thị giao diện web trên các thiết bị.

Như đã nói ở mục Responsive Web Design, các giao diện web khi được thiết kế bằng bootstrap sẽ có độ tương thích với tất cả các thiết bị. Đây chính là ưu điểm nổi bật được sử dụng nhiều nhất của bootstrap.

Bootstrap khiến giao diện web tương thích với tất cả các thiết bị

Cách sử dụng bootstrap

Để hiểu được cách sử dụng bootstrap, hãy xem ví dụ dưới đây.

Mã hóa ký tự cho tài liệu HTML. Trong trường hợp này, UTF-8 đề cập đến Unicode.

meta charset="utf-8"

Cho biết bộ ký tự được sử dụng để viết trang web.

meta http-equiv="X-UA-Compatible"

Xác định phiên bản Internet Explorer sẽ hiển thị trang. Sử dụng chế độ Edge, nó được cài đặt để sử dụng chế độ cao nhất hiện có.

meta name="viewport"

Đảm bảo rằng trang có tỷ lệ 1: 1 với kích thước khung nhìn.

link href="css/bootstrap.min.css" rel="stylesheet"

Đây là phần thêm Bootstrap core CSS.

src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"

Tải jQuery thông qua Google CDN. Sẽ tốt hơn để bạn tải nó từ CDN qua HTTP vì file có thể được lưu trong bộ nhớ cache trong một năm.

src="js/bootstrap.min.js

Thêm JavaScript lõi Bootstrap. Cú pháp này phải luôn nằm dưới cú pháp jQuery để hoạt động đúng. Quá trình bổ sung có thể được thực hiện thông qua URL của Google hoặc tải xuống thủ công.F
